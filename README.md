#  AI-Powered Lost & Found System  

An intelligent web-based Lost & Found management system that uses **AI logic** to help match found items with possible owners based on description similarity and confidence scores.  

---

##  Features
- Report lost or found items easily through an interactive web interface  
- AI generates suggestions of possible owners with confidence levels  
- View all reported items in an organized format  
- Clean and responsive design using HTML, CSS, and Flask Jinja templates  
- Stores data securely in MySQL database  

---

## Technologies Used
- **Frontend:** HTML, CSS, Jinja Templates  
- **Backend:** Python (Flask)  
- **Database:** MySQL  
- **AI Component:** Simple confidence-based matching algorithm  

---

##  How It Works
1. A user reports a *lost* or *found* item through the app.  
2. The system stores the data and analyzes item descriptions.  
3. AI compares new entries with existing data to suggest possible matches.  
4. Matches are displayed with confidence percentages.
