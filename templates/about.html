About This Project

The AI Lost & Found Portal is a web-based system developed using Python (Flask) and MySQL that helps users to report and locate lost or found items efficiently. This platform aims to bridge the gap between individuals who have lost their belongings and those who have found unclaimed items.

The portal uses a simulated Artificial Intelligence (AI) logic that analyzes the description of a found item and predicts the most likely owner‚Äôs name with a confidence score. This represents how AI can assist humans in making smart, data-driven decisions in real-world applications.

Users can report both found items and lost items with details such as item name, description, and category. All records are stored securely in a MySQL database, making the process organized and easy to track. The system demonstrates how database management, web development, and AI logic can work together to create a practical solution.
